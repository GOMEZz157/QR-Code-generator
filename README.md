# 🔳 QR Code Generator

Gerador de QR Code desenvolvido com **HTML, CSS e JavaScript**, consumindo uma **API pública de QR Code** para gerar códigos dinamicamente a partir de qualquer texto ou link.

🔗 Repositório:  
https://github.com/GOMEZz157/QR-Code-generator

---

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- Manipulação do DOM
- Consumo de API externa
- Estruturação semântica com HTML
- Estilização responsiva com CSS
- Organização de lógica com JavaScript puro (Vanilla JS)

O usuário pode inserir qualquer texto ou URL e gerar instantaneamente um QR Code correspondente.

---

## 🚀 Funcionalidades

- ✅ Geração dinâmica de QR Code
- ✅ Atualização em tempo real
- ✅ Validação para evitar input vazio

- ✅ Layout responsivo

## ⚙️ Como Executar

Acesse [https://qr-code-generator-lilac-phi-90.vercel.app/](https://qr-code-generator-lilac-phi-90.vercel.app/)

ou:

1. Clone o repositório:

```bash
git clone https://github.com/GOMEZz157/QR-Code-generator.git
```

2. Acesse a pasta do projeto:

```bash
cd QR-Code-generator
```

3. Abra o arquivo `index.html` no navegador.

Não é necessário instalar dependências.
